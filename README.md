# bandit-solutions
Solutions and short descriptions for each level in wargame called "Bandit".
https://overthewire.org/wargames/bandit
The solutions may not be entirely accurate, so this shouldn't be considered a tutorial. Likewise, the explanations—I haven't tried to describe everything professionally, and there may be some mistakes. This is more of a collection of my notes and thoughts on each level. Anyone who stumbles upon this repository should keep that in mind.
